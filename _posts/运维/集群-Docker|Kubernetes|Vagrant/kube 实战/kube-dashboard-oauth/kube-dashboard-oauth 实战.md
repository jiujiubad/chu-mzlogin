## 前提

原来的 kubernetes 需要运行 `kubectl proxy` 即暴露 8001 端口，才能访问 Dashboard 页面。

使用 kube-dashboard-oauth，把 Dashboard 绑定在域名 dashboard.staging.muker.io 上，通过第三方登录验证（项目选用了 Gitlab）。
