### 1】utf8 error报错

```
vim ~/.zshrc
```

在最后填写

```
export LC_ALL=en_US.UTF-8
export LANG=en_US.UTF-8
```

## 3、编码

```
export LANGUAGE=en_US.UTF-8
export LANG=en_US.UTF-8
export LC_ALL=en_US.UTF-8
sudo locale-gen en_US.UTF-8
sudo dpkg-reconfigure locales
```
